&lt;bare-select&gt;
===================

**A stylable replacement for &lt;select&gt;.**  
**Scripts from us.**  
**Styles by you.**




Installation
------------

```sh
$ npm install git@git.sb12.de:js/lib/bare-select
$ cd node_modules/bare-select
$ npm install
$ npm run prepublish
```


Once we’ve published to npmjs.org it’ll be a lot easier:

```sh
$ npm install bare-select
```
